# MediCcite - Sistema de Gestión de Citas Médicas
Proyecto Final - Programación III - ITLA

## Descripción
Sistema web para la gestión de citas y pacientes (MVP).

## Tecnologías
- Frontend: HTML5, CSS3, JavaScript.
- Pruebas: Selenium con Python (ver archivo `test_mediccite.py`).

## Cómo ver el proyecto
Descargue los archivos y abra `index.html` en su navegador.
